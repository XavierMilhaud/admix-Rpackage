# Dataset of heliocentric velocity for the Milky Way

Dataset of heliocentric velocity for the Milky Way

## Usage

``` r
milkyWay
```

## Format

A data frame with 170,601 rows and 1 column:

- V1:

  Heliocentric velocity measurements of the Milky way
