# Measurements of heliocentric velocities

An evolving data frame of velocities for 4 dSph galaxies (namely Carina,
Sextans, Sculptor and Fornax), from SIMBAD astronomical database.

## Usage

``` r
allGalaxies
```

## Format

Currently contains 8,862 rows and 3 columns, with information on:

- Target:

  Target identification; Galaxy-ID number

- HV:

  Weighted mean Heliocentric rest frame velocity

- Name:

  The name of the galaxy
