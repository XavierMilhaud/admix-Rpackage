# Print method for object of class 'admix_cluster'

Print the main results when clustering the unknown component
distributions coming from various admixture samples, i.e. the obtained
clusters.

## Usage

``` r
# S3 method for class 'admix_cluster'
print(x, ...)
```

## Arguments

- x:

  An object of class 'admix_cluster' (see ?admix_clustering).

- ...:

  further arguments passed to or from other methods.
