# Extractor for components involved in test statistic

Provide the terms (or discrepancies) that compose the test statistic for
the k-sample test.

## Usage

``` r
get_statistic_components(x)
```

## Arguments

- x:

  An object of class `IBM_test`.

## Value

The components finally included in the test statistic, i.e. the
discrepancies of the couples that were aggregated in the built sequence
of statistics.
