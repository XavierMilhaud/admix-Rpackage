# Print method for objects `twoComp_mixt`

Print an object of class `twoComp_mixt`. A two-component mixture model
has probability density function (pdf) l such that: l = p \* f + (1-p)
\* g, where p is the mixing proportion, and f and g are the component
distributions.

## Usage

``` r
# S3 method for class 'twoComp_mixt'
print(x, ...)
```

## Arguments

- x:

  An object of class `twoComp_mixt`.

- ...:

  A list of additional parameters belonging to the default method.
